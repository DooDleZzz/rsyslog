$ModLoad

Type: global configuration directive

Default:

Description:

This currently is a dummy directive. It will support the loading of plug-ins in future releases of rsyslog supporting plug-ins. Currently, only "MySQL" is supported as parameter. This activates MySQL support (if rsyslog is compiled with MySQL functionality).

Sample:

$ModLoad MySQL
